# PWA para Android: Introdução e Conceitos Fundamentais

## Introdução
Uma descrição abrangente do que esta skill abordará, seu propósito fundamental, sua relevância estratégica no contexto do desenvolvimento web/programação e como ela se interconecta com outras áreas do conhecimento. Deve ser um panorama completo do que a IA pode esperar aprender.

As **Progressive Web Apps (PWAs)** representam a convergência definitiva entre a web e o mobile. No Android, essa tecnologia atinge seu ápice devido à integração profunda com o sistema operacional através de WebAPKs. Esta skill explora como transformar sites em aplicativos instaláveis, rápidos e resilientes.

## Glossário Técnico
*   **PWA (Progressive Web App)**: Aplicativo web que utiliza tecnologias modernas para oferecer experiência nativa.
*   **WebAPK**: Formato de empacotamento que o Android usa para instalar PWAs como apps reais.
*   **App Shell**: Estrutura mínima de UI necessária para o app carregar instantaneamente.

## Conceitos Fundamentais
A filosofia "Progressive" garante que o app funcione em qualquer lugar, mas brilhe em dispositivos modernos. O modelo mental correto é pensar na web como uma plataforma de entrega de software, não apenas de documentos.

### Subtópico 1.1: A Experiência Nativa no Android
No Android, uma PWA instalada aparece na gaveta de apps, gerencia notificações nativas e pode ser definida como o app padrão para certas ações. Isso é possível graças ao **WebAPK**, que "engana" o sistema fazendo-o acreditar que a PWA é um app nativo.
